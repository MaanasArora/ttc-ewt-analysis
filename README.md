# TTC Excess Wait Time Analysis

Very rough WIP to analyze GTFS real-time data from the Toronto Transit Commission (TTC) to calculate excess waiting time (EWT) at stops. The EWT reflects how much longer passengers generally wait because of irregularity (bunching) in the bus schedule, regardless of the scheduled headway:

Excess Waiting Time = Actual Waiting Time Because Bus is Bunched - Waiting Time if Bus Is Perfectly Regular
